# 📚 刷题助手

一个简洁的手机刷题应用，支持单选题和多选题，可在手机浏览器中使用。

## 🚀 快速开始

1. 直接双击 `index.html` 文件在浏览器中打开
2. 或者用手机扫描二维码访问（需要先部署到服务器）

## 📝 如何导入题库

### 题库格式说明

打开 `questions.js` 文件，按照以下格式添加您的题目：

```javascript
window.QUESTIONS = [
    // 单选题格式
    {
        id: 1,                    // 题目ID（必填，唯一）
        type: "single",           // 题目类型（必填）: "single"单选 或 "multiple"多选
        category: "分类名称",      // 分类（可选）
        question: "题目内容？",    // 题目内容（必填）
        options: [                // 选项数组（必填）
            "选项A",
            "选项B", 
            "选项C",
            "选项D"
        ],
        answer: 0,                // 正确答案索引（必填，从0开始，A=0, B=1, C=2, D=3）
        explanation: "答案解析"    // 解析（可选）
    },
    
    // 多选题格式
    {
        id: 2,
        type: "multiple",         // 多选题
        category: "分类名称",
        question: "题目内容？（多选）",
        options: [
            "选项A",
            "选项B", 
            "选项C",
            "选项D"
        ],
        answer: [0, 2],           // 多选答案是数组，如AC=[0, 2]，ABD=[0, 1, 3]
        explanation: "答案解析"
    }
];
```

### 从Excel导入题目

如果您的Excel格式如下：

| 题号 | 题型 | 分类 | 题目 | 选项A | 选项B | 选项C | 选项D | 答案 | 解析 |
|-----|-----|-----|-----|------|------|------|------|-----|-----|
| 1 | 单选 | 法律 | 问题内容 | 选项1 | 选项2 | 选项3 | 选项4 | A | 解析内容 |
| 2 | 多选 | 法律 | 问题内容 | 选项1 | 选项2 | 选项3 | 选项4 | ABC | 解析内容 |

**转换步骤：**

1. 在Excel中，答案列使用字母（A、B、C、D或ABC、ABD等组合）
2. 将Excel另存为CSV格式
3. 使用项目中的 `convert.html` 工具进行转换（如果有）
4. 或者手动按照上述格式编辑 `questions.js`

**答案转换对照表：**
- A → 0
- B → 1
- C → 2
- D → 3
- AB → [0, 1]
- AC → [0, 2]
- ABC → [0, 1, 2]
- 以此类推...

## ✨ 功能特性

- 📱 **手机适配** - 响应式设计，完美适配各种屏幕
- 🎲 **多种模式** - 顺序刷题、随机刷题、错题重做、分类练习
- 💾 **进度保存** - 自动保存学习进度，下次继续
- 🌙 **深色模式** - 自动适配系统深色模式
- 📊 **统计分析** - 实时显示正确率和完成进度

## 🔧 自定义

### 修改主题颜色

编辑 `style.css` 文件开头的 CSS 变量：

```css
:root {
    --primary-color: #4f46e5;    /* 主题色 */
    --success-color: #10b981;    /* 正确颜色 */
    --error-color: #ef4444;      /* 错误颜色 */
}
```

## 📱 部署到手机使用

### 方法1：本地网络访问
1. 在电脑上启动一个简单的HTTP服务器：
   ```bash
   # 如果有Python
   cd quiz-app
   python3 -m http.server 8080
   ```
2. 手机和电脑连接同一WiFi
3. 手机浏览器访问 `http://电脑IP:8080`

### 方法2：部署到GitHub Pages（免费）
1. 创建GitHub仓库
2. 上传所有文件
3. 在仓库Settings中启用GitHub Pages
4. 通过生成的链接访问

### 方法3：部署到其他静态托管服务
- Vercel
- Netlify
- Cloudflare Pages

## ❓ 常见问题

**Q: 为什么打开是空白？**
A: 请检查 `questions.js` 文件格式是否正确，特别注意逗号和引号。

**Q: 进度如何重置？**
A: 点击首页底部的"重置学习进度"按钮。

**Q: 支持多少道题？**
A: 理论上没有限制，但建议不超过1000道以保证流畅性。
